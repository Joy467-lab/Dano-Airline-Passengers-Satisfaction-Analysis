# Dano Airline Passenger Satisfaction Analysis

## Overview

This project was developed using Power BI to analyze passenger satisfaction for Dano Airlines. The dashboard helps identify customer satisfaction levels, service performance, and factors that influence passenger experience.

## Tools Used

- Power BI
- Power Query
- DAX

## Dataset

The dataset contains information about airline passengers, including:

- Customer type
- Gender
- Age
- Travel class
- Travel type
- Flight distance
- Arrival delay
- Service ratings
- Passenger satisfaction

## Data Preparation

Before building the dashboard, the data was prepared by:

- Checking for missing values
- Removing duplicates
- Setting the correct data types
- Creating Age Group and Delay Category columns
- Creating DAX measures for the KPIs

## Dashboard Features

The dashboard includes:

- Total Passengers
- Satisfaction Rate
- Dissatisfaction Rate
- Average Arrival Delay
- Average Flight Distance
- Passenger Satisfaction Breakdown
- Satisfaction by Customer Type
- Satisfaction by Age Group
- Satisfaction by Travel Class
- Average Service Ratings
- Average Arrival Delay by Satisfaction

## Key Findings

- More than half of the passengers were neutral or dissatisfied.
- Longer arrival delays are associated with lower passenger satisfaction.
- Business Class passengers were generally more satisfied than Economy passengers.
- In-flight WiFi and Ease of Online Booking received the lowest service ratings.

## Recommendations

- Improve the quality of in-flight WiFi.
- Reduce arrival delays to improve customer satisfaction.
- Improve the online booking process.
- Continue maintaining high standards for baggage handling and seat comfort.
